================================================================================
     새 PC에서 SAP 데이터 포털 설치하기 (완전 처음부터)
================================================================================

【 순서 요약 】
1단계: Python 설치
2단계: Node.js 설치
3단계: Claude Code 설치
4단계: Claude에게 복사/붙여넣기로 자동 설치 요청

================================================================================
【 1단계: Python 설치 】
================================================================================

1. 크롬 브라우저 열기

2. 주소창에 입력:
   python.org

3. 파란색 "Downloads" 클릭

4. "Download Python 3.x.x" 노란 버튼 클릭

5. 다운로드된 파일 실행

6. ★★★ 중요 ★★★
   설치 화면 맨 아래에 있는 체크박스:

   [v] Add Python to PATH   ← 이거 반드시 체크!!!

7. "Install Now" 클릭

8. 설치 완료될 때까지 기다리기

================================================================================
【 2단계: Node.js 설치 】
================================================================================

1. 크롬 브라우저에서 주소창에 입력:
   nodejs.org

2. 왼쪽 초록색 버튼 "LTS" 클릭 (다운로드됨)

3. 다운로드된 파일 실행

4. 계속 "Next" 클릭하면서 설치

5. 마지막에 "Finish" 클릭

================================================================================
【 3단계: Claude Code 설치 】
================================================================================

1. 키보드에서 Windows키 + R 동시에 누르기

2. 나오는 창에 입력:
   cmd

3. 엔터 누르기 (검은 창이 열림)

4. 검은 창에 아래 내용 복사해서 붙여넣기:

   npm install -g @anthropic-ai/claude-code

5. 엔터 누르고 설치될 때까지 기다리기 (1-2분)

================================================================================
【 4단계: Claude 실행하고 자동 설치 요청하기 】
================================================================================

1. 검은 창(명령 프롬프트)에 입력:

   cd D:\유은상\test\sap-screen

   (폴더 경로는 실제 복사한 위치로 변경)

2. Claude 실행:

   claude

3. API 키 입력하라고 나오면:
   - https://console.anthropic.com/ 접속
   - 회원가입/로그인
   - "API Keys" 메뉴 클릭
   - "Create Key" 클릭
   - 생성된 키 복사해서 붙여넣기

4. Claude가 실행되면 아래 내용을 통째로 복사해서 붙여넣기:

================================================================================
【 ★★★ 이 아래 내용을 Claude에게 복사/붙여넣기 ★★★ 】
================================================================================

이 프로젝트는 SAP 데이터 포털이야. 새 PC에 처음 설치하는 거라서 필요한 패키지들을 설치해줘.

아래 명령어를 실행해서 설치해줘:

pip install flask pandas openpyxl python-dateutil

설치가 완료되면 서버를 실행해줘:

python app.py

그리고 브라우저에서 http://localhost:5001 열어줘

================================================================================
【 ★★★ 복사/붙여넣기 끝 ★★★ 】
================================================================================

5. Claude가 자동으로:
   - 필요한 패키지 설치
   - 서버 실행
   - 브라우저 열기

   를 해줌!

================================================================================
【 설치 완료 후 매일 사용할 때 】
================================================================================

방법 1: 배치 파일 사용 (쉬움)
   → "SAP포털실행.bat" 더블클릭

방법 2: Claude와 함께 사용
   → "2_Claude실행.bat" 더블클릭
   → "서버 실행해줘" 라고 말하기

================================================================================
【 문제가 생겼을 때 Claude에게 할 말 】
================================================================================

오류가 나면 이렇게 말해:

"이런 오류가 나는데 해결해줘:
[오류 메시지를 여기에 복사/붙여넣기]"

================================================================================
【 API 키 발급 방법 (상세) 】
================================================================================

1. 크롬에서 주소창에 입력:
   console.anthropic.com

2. "Sign Up" 클릭해서 회원가입
   (구글 계정으로 가입 가능)

3. 로그인 후 왼쪽 메뉴에서 "API Keys" 클릭

4. "Create Key" 버튼 클릭

5. 이름 입력 (아무거나 예: "mykey")

6. "Create Key" 클릭

7. 나오는 키(sk-ant-xxx...)를 복사

8. Claude Code에서 키 입력하라고 하면 붙여넣기

※ 키는 한 번만 보여주니까 꼭 어딘가에 저장해두기!
